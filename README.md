## 🎯 Project Overview

This project demonstrates the use of **Inheritance in Object-Oriented Programming (OOP)** using Python.  
A base class `Vehicle` is extended by child classes `Car`, `Bike`, and `Truck`.

---

## ✨ Features

| Feature | Description |
|--------|------------|
| ➕ Add Vehicle | Add Car, Bike, or Truck |
| 📋 View Vehicles | Display all vehicle details |
| 🧬 Inheritance | Base class reused in child classes |
| 🔁 Method Overriding | Each vehicle shows custom info |
| 🔁 Menu System | Interactive CLI |

---

## 🧠 How It Works

Vehicle (Base Class)  
↓  
Car / Bike / Truck (Child Classes)  
↓  
User Input → Menu → Operation  
↓  
Vehicle Objects Stored in List  
↓  
Display using Polymorphism
